## 🚀 How to Use Quill

### Run a Program
```bash
# Windows
quill program.quill

# macOS/Linux  
./quill program.quill

# Or directly
python core/quill.py program.quill
```

### Try Examples
```bash
quill examples/example_simple.quill
quill examples/example_guessing_game.quill
quill examples/example_calculator.quill
quill examples/example_adventure.quill
```
